Lesson 07 · iOS

Xcode 的秘密

第一次进入 iOS 开发环境，用 SwiftUI、API 调用和应用签名做出自己的 App。

时长约 80 分钟
产出iOS App
准备Mac、Xcode、Apple ID

这一节还没有解锁

请先完成前一节，再进入本节。

学习目标

理解 iOS 项目如何被组织和运行。

  • 认识 Xcode、SwiftUI 和预览窗口。
  • 让 App 调用一个真实 API。
  • 理解签名、模拟器和真机调试的差异。
课堂流程

从 SwiftUI 页面到可运行 App

  1. 创建最小 iOS 项目。
  2. 生成界面和状态逻辑。
  3. 接入 API 并处理加载状态。
  4. 检查签名和运行环境。
课后作业

提交 App 演示

提交模拟器或真机录屏，以及一份遇到的 Xcode 报错与修复记录。
