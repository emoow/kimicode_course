Lesson 02 · Agent

造一个你自己的简易版 Manus

从“让 AI 做一件事”升级到“让 agent 规划、调用 skill、完成复杂任务”。

时长约 75 分钟
产出会规划并调用多个 skill 的 agent
准备IDE + CLI 配置

这一节还没有解锁

请先完成前一节，再进入本节。

学习目标

理解 agent = 会规划的大脑 + 一堆 skill。

  • 区分一次性对话和可规划的 agent。
  • 至少准备 3 个 skill。
  • 写出 orchestration prompt，让 agent 先规划再执行。
课堂流程

从总任务到自动交付

  1. 选择一个必须拆步骤的大目标。
  2. 准备调研、写报告、做幻灯片、统一风格等 skill。
  3. 写 agent 主指令，要求它逐步执行并汇总产出。
  4. 选做接 MCP，让 agent 能读写真实文件。
课后作业

提交你的迷你 Manus

提交 3 个 skill、一次完整运行结果，以及一份运行记录：拆了几步、用了什么 skill、踩了什么坑。
