Lesson 03 · Next.js

把 Manus 做成真正的项目

把 agent 从聊天记录搬到网页项目里，第一次接触版本管理、运行时、框架和密钥管理。

时长约 75 分钟
产出接真实 API 的 Next.js 网页
准备GitHub 账号

这一节还没有解锁

请先完成前一节，再进入本节。

学习目标

让 agent 成为真正属于你的网页应用。

  • 用 Git 给项目建立存档点。
  • 认识 Node.js、Next.js 和本地开发服务器。
  • 用 .env 保存 API key，并确认不会上传到 GitHub。
课堂流程

从本地跑通到接入 API

  1. 初始化 Git 并推到 GitHub。
  2. 让 KimiCode 创建最小 Next.js 项目。
  3. 加入输入框、后端 API 调用和结果展示。
  4. 检查 .env 与 .gitignore，建立安全红线。
课后作业

提交项目链接和运行截图

提交 GitHub 链接、本地运行截图，以及确认 .env 未上传的安全清单。
