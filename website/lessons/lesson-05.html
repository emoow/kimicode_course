Lesson 05 · Extension

做一个 Chrome 扩展

把 AI 协作从网页项目迁移到浏览器扩展，做一个能装进自己浏览器的小工具。

时长约 75 分钟
产出Chrome 扩展
准备无需额外准备

这一节还没有解锁

请先完成前一节，再进入本节。

学习目标

理解浏览器扩展的基本结构。

  • 认识 manifest、content script 和 popup。
  • 让扩展读取当前页面上下文。
  • 在浏览器里加载并调试本地扩展。
课堂流程

从文件结构到装进浏览器

  1. 确定一个日常浏览器场景。
  2. 生成 manifest 和 popup 页面。
  3. 添加 content script 与权限声明。
  4. 加载未打包扩展并修复报错。
课后作业

提交可安装的小工具

提交扩展文件夹、安装截图和一次真实使用录屏。
