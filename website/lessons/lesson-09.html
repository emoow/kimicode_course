Lesson 09 · Self Improving Agent

AI 的自我进化

用 OpenClaw、prompt 改写、反馈循环和自我评估机制，做一个会根据反馈持续优化自己的 agent 系统。

时长约 80 分钟
产出会自我优化的 agent 系统
准备Python 环境、前两节基础

这一节还没有解锁

请先完成前一节，再进入本节。

学习目标

让 agent 在反馈中改进自己。

  • 理解反馈循环、自我评估和 prompt 改写。
  • 认识 OpenClaw 框架的基本角色。
  • 建立一套可观察、可回滚的优化流程。
课堂流程

从一次反馈到持续优化

  1. 选择一个前面做过的 agent。
  2. 设计评分标准和反馈输入。
  3. 让系统提出 prompt 改写建议。
  4. 比较优化前后的输出质量。
课后作业

提交自我优化系统

提交运行记录、优化前后对比、失败案例和下一轮改进计划。
