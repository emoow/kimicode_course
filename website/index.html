核心命题

非技术背景的人，怎么把想法做成产品？

传统编程让你先背语法和术语，才敢动手。我们反过来：你先把想法说清楚， KimiCode 帮你做第一版，你在真实成品里理解它是怎么回事。

先有成品，再懂原理

第一节就做 3D 月球网页，用一个足够具体的 prompt 打掉心理门槛，然后再拆解为什么它有效。

做的是真产品

课程覆盖网页、Agent、Next.js、舆情看板、Chrome 扩展、小程序、iOS App 和部署上线。

报错就是教材

命令记不住没关系，卡住了就把报错丢给 KimiCode，让它教你修。会求助、会验收，比硬背更重要。

课程地图

从一句话造月球，到把产品送上线

九节课按能力递进：先获得 wow moment，再学会造 agent、接真实 API、做不同产品形态，最后部署和自我优化。

贯穿全程的一条原则：命令记不住没关系，遇到报错就截图丢给 KimiCode，让它教你修。

怎么学

拉仓库、看演示、跟着做、交作业

异步自主学习，按自己的节奏来。每节课有演示视频、教程文档、作业提交和验收反馈。

01

拉仓库到本地

把课程仓库 git clone 下来，拿到课程网页、配套 skill、AGENTS.md 和每节作业说明。

02

看演示再动手

每节先看成品演示，再跟教程一步步做。先跑通，再回头理解背后的概念。

03

和 KimiCode 协作

用 prompt、skill 和 agent 把复杂目标拆开，让 AI 规划、执行、修错、整理交付。

04

提交成果验收

提交截图、录屏、GitHub 链接或运行记录。讲师和助教按清单反馈，讨论区 24 小时内接住卡点。

学习成果

每一节课都对应一个能展示的交付物

课程不是概念课。每个主题都落到可运行、可截图、可提交、可继续迭代的成果。

01

3D 可交互月球网页

用一句有效 prompt 做出第一个 wow moment。

02

会规划、会调用 skill、能完成多步骤任务。

03

接真实 API 的 Next.js 项目

理解 Git、Node、Next、.env 和 API 调用。

04

定时舆情分析看板

登录、数据库、定时任务和官方 API 合规使用。

05

Chrome 扩展

真正装进浏览器使用的小工具。

06

微信小程序

能在模拟器跑通，并理解审核流程。

07

iOS App

用 SwiftUI、API 和签名做出自己的 App。

08

上线作品合集

把前面项目送到真实用户可以打开的位置。
