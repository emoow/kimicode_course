你不需要先学编程。你有领域里的洞察、有审美、有真实的需求——把它描述清楚， KimiCode 帮你做完所有技术决定。这是一门写给非技术背景的人的 Vibe Coding 课。

核心命题

非技术背景的人，怎么把想法做成产品？

传统编程让你先背一大堆语法和术语，才敢动手。我们反过来：你说清楚想要什么， KimiCode 把它做出来，你在真实的成品里理解它是怎么回事。技术选型不用你操心。

有个懂技术的向导

我们写了一套「Weakerpowers」skill——一个友好的技术向导，替你做所有技术决定，只跟你解释「为什么」，绝不让你回答「用什么框架」。

做的是真产品

不是练习题。基础篇用一个完整的个人网站，把前端、后端、部署全部串起来，做完就是一个能用、能发给别人的真实产品。

报错不可怕

命令记不住没关系，卡住了就把报错丢给 KimiCode，让它教你修。学会和 AI 一起解决问题，比记住命令更重要。

课程地图

从一句话上线，到做出不同形态的产品

先用 90 分钟跑通一次完整闭环，再用一个个人网站项目逐层加能力，最后按兴趣做不同形态的产品。

贯穿全程的一条原则：命令记不住没关系，遇到报错就截图丢给 KimiCode，让它教你修。

怎么学

拉仓库、读教程、和 KimiCode 一起做作业

异步自主学习，按自己的节奏来。卡住了，公开的作业与讨论板块随时有人接住你。

01

拉仓库到本地

把课程仓库 git clone 下来，里面有每节配套的 skill、告诉 KimiCode 当前是课程环境的 AGENTS.md，以及每道作业的说明。

02

读教程跟着做

教学网站承载每节课的正文。跟着逐步指引走，不追求一上来就理解，先跑通，再慢慢懂原理。

03

用 skill + KimiCode

每个 skill 给 KimiCode 装上一项专项能力。用 brainstorming 细化需求，用前端 skill 出设计，向导替你拍板技术。

04

发作业到社群

把成品发到公开的作业展示板块，晒作业、互相点评、答疑。优秀作业会被定期精选展示。

适合谁

这门课是为你准备的

  • 非理工科背景，没写过一行代码，但学习能力强
  • 在自己的领域有 know-how，知道什么值得做
  • 有想法、有需求洞察，想把它做成真东西
  • 不打算转行做开发，想把「用 AI 实现想法」当职场技能
